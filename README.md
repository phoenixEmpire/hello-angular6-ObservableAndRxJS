# HelloAngular6ComponentAndTemplate

Angular6 Observable&RxJS.

## 安装

运行 npm install 命令.

## 启动

运行 ng serve --open .

## 浏览器

Chrome开启跨域.

Safari停止跨源限制.

## 升级到Angular6

1 sudo npm install -g @angular/cli@latest (Mac环境)

2 npm install @angular/cli@latest --save-dev

3 ng update @angular/cli --migrate-only --from=1.7.4
